# Project Memory

## Core
Cali: Mobile-first social planning app for effortless people-first coordination. Always refer to the app as "Cali" (never SyncCircle/SyncCycle).
Tech stack: Lovable Cloud, Supabase Auth + Relational DB.
Design: Warm coral & cream palette, DM Sans. Clean, avatar-driven UI, rounded cards, Framer Motion animations.

## Memories
- [Home Screen](mem://features/home-screen) — Home dashboard with scorecard, upcoming plans, and pending RSVPs
- [Calendar Availability](mem://features/calendar-availability) — Friend availability overlay via @mention on the calendar grid
- [Calendar Views](mem://features/calendar-views) — Month, Week, and Day view modes with inline event creation shortcuts
- [Event Creation](mem://features/event-creation) — Single-page form for event creation, supporting time ranges and search-based friend @-mentions
- [Event Social Features](mem://features/event-social) — Attendee list, RSVP statuses, comments, @mentions, and shared photo gallery
- [Friend Management](mem://features/friend-management) — Request-and-accept system with fuzzy search and notification badges
- [Activity Feed](mem://features/activity-feed) — Centralized feed for event invitations, comments, and @mentions
- [User Profiles & Onboarding](mem://features/user-profiles) — User setup capturing username, avatar, and social bio, managed in Settings
- [Supabase RLS Architecture](mem://auth/backend-architecture) — SECURITY DEFINER functions for event and participant data operations
